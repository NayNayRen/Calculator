Calculate Some Numbers

A calculator done by me.

Why? Because...why not? Yeah, there's calculators in literally everything now, but I didn't build those. I built this one. It can perform some basic math, add, subtract, multiply and divide.

If you click the input on desktop, or tap on mobile, you can use parenthesis for precedence. If something doesn't evaluate, it will tell you.
